# 코드
---

```
a=int(input())
b=input()
c=int(input()) 
if b=="+":
    print(a+c)
else:
    print(a*c)
```

# 풀이
---
1. 숫자, 연산자를 세줄로 각각 입력받는다.

2. if 문으로 연산한 후 출력한다.

# 생각
---
